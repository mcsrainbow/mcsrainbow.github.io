我是个菜的不能再菜的人！

<!--more-->

我在论坛求助：一无所获！

“apt-get -f install 无法通过，望前辈们帮忙解决！”

“在网上找了很多，也想了很多办法，还是不行，希望得到大家的帮助，谢谢了！”

“我希望通过apt-get install 来安装一些软件，但是我安装几乎任何软件都提示我cpp c++等依赖性的问题，并且提示我使用apt-get -f install来修正这些依赖性，但是我一旦执行该命令就会出现这个错误。”

我在网上寻找资料：一无所获！

时间这样被拖延下来，我视乎只能去做一些别人已经做好的事情，我只需要拿来照着做。我真的太缺少变通的能力，一个这样的问题不应该卡住一个人的，人总应该学会变通的。

我有自己的计划，应该整理自己的思绪了，把计划重新整理好，我应该让自己成长起来，不要再这样菜了！
