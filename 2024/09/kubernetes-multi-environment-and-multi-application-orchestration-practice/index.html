Manifests

еҺҹз”ҹ YAML(Manifests) жҳҜжңҖзӣҙи§Ӯе’ҢжңҖз®ҖеҚ•зҡ„ Kubernetes иө„жәҗй…ҚзҪ®ж–№ејҸпјҢеҜ№дәҺеӨҡзҺҜеўғеӨҡеә”з”Ёзҡ„зј–жҺ’ж–№ејҸд№ҹз®ҖеҚ•зІ—жҡҙпјҢзӣҙжҺҘеҲӣе»әдёҚеҗҢзҺҜеўғеҗҚз§°зҡ„зӣ®еҪ•пјҢе°Ҷиө„жәҗжҢүз…§е…¬е…ұе’Ңеә”з”Ёз»ҙеәҰеҲҶеҲ«еҶҷе…ҘдёҚеҗҢзҡ„ YAML ж–Үд»¶еҚіеҸҜгҖӮ

дјҳзӮ№: жңҖзӣҙи§ӮпјҢдёҚеҗҢзҺҜеўғе’Ңеә”з”Ёд№Ӣй—ҙй…ҚзҪ®ж— иҖҰеҗҲпјҢй…ҚзҪ®еӨұиҜҜж—¶еҪұе“ҚиҢғеӣҙе°Ҹпјӣ
зјәзӮ№: йҮҚеӨҚзҡ„й…ҚзҪ®д»Јз ҒиҫғеӨҡпјҢдҝ®ж”№й…ҚзҪ®еҸӮж•°ж—¶дёҚж–№дҫҝпјҢйңҖиҰҒйҖҡиҝҮж–Үжң¬ж–№ејҸжҹҘжүҫе’ҢжӣҝжҚўгҖӮ

Kustomize

Kustomize жҳҜд»ӢдәҺ YAML е’Ң Helm д№Ӣй—ҙзҡ„иө„жәҗй…ҚзҪ®ж–№ејҸпјҢеңЁе…је®№ YAML зҡ„еҹәзЎҖдёҠйҖҡиҝҮеҶ…зҪ®зҡ„жҸ’д»¶е®һзҺ°й…ҚзҪ®д»Јз Ғзҡ„еӨҚз”Ёе’ҢеҸӮж•°зҡ„дҝ®ж”№гҖӮ

Kustomize й»ҳи®ӨиҜ»еҸ– kustomization.yaml ж–Үд»¶дёӯзҡ„й…ҚзҪ®пјҢеҸҜйҖҡиҝҮеј•з”ЁдёҚеҗҢзӣ®еҪ•дёӢзҡ„ kustomization.yaml ж–Үд»¶е®һзҺ°еҲҶеұӮи®ҫи®ЎгҖӮ

cd kustomize/apps-overlays
kustomize build overlays/development
kustomize build overlays/staging
kustomize build overlays/production

дјҳзӮ№: е…је®№еҺҹз”ҹ YAMLпјҢеҸҜеӨҚз”Ёй…ҚзҪ®д»Јз ҒпјҢеҸҜйҖҡиҝҮеҶ…зҪ®зҡ„жҸ’д»¶иҜ»еҸ–ж–Үд»¶з”ҹжҲҗ Secret е’Ң ConfigMapпјҢд»ҘеҸҠдҝ®ж”№еҸӮж•°пјҲдҫӢеҰӮ: еўһеҠ ж ҮзӯҫгҖҒеўһеҠ иө„жәҗеҗҚз§°еүҚзјҖгҖҒдҝ®ж”№ CPU е’ҢеҶ…еӯҳзӯүпјүпјӣ
зјәзӮ№: еӨҚз”Ёй…ҚзҪ®д»Јз ҒеҗҺд№ҹеҜјиҮҙдәҶиҖҰеҗҲпјҢй…ҚзҪ®еӨұиҜҜж—¶еҪұе“ҚиҢғеӣҙеўһеӨ§пјҢеҶ…зҪ®зҡ„жҸ’д»¶еҠҹиғҪжңүйҷҗпјҢж— жі•й«ҳеәҰиҮӘз”ұең°еӨҚз”Ёй…ҚзҪ®д»Јз Ғе’Ңдҝ®ж”№еҸӮж•°гҖӮ

Helm

Helm жҳҜ Kubernetes еҢ…з®ЎзҗҶеҷЁе’ҢйғЁзҪІе·Ҙе…·пјҢзұ»дјј Linux дёҠзҡ„ й…ҚзҪ®з®ЎзҗҶе·Ҙе…· Ansible + еҢ…з®ЎзҗҶеҷЁ YumгҖӮ

Helm й»ҳи®ӨиҜ»еҸ– values.yaml ж–Үд»¶дёӯе®ҡд№үзҡ„еҸҳйҮҸпјҢе’Ң templates зӣ®еҪ•дёӢзҡ„жЁЎжқҝж–Үд»¶гҖҒеҮҪж•°е’Ңд»Јз Ғеқ—пјҢеҸҜд»Ҙй«ҳеәҰиҮӘз”ұең°еӨҚз”Ёд»Јз ҒгҖҒдҝ®ж”№еҸӮж•°е’ҢиҜ»еҸ–ж–Үд»¶з”ҹжҲҗй…ҚзҪ®гҖӮ

Helm еҸҜд»ҘеҲӣе»әдёҚеҗҢзҺҜеўғзҡ„ values ж–Үд»¶пјҢе°Ҷй»ҳи®ӨеҸӮж•°е’Ңе…¬е…ұеҸӮж•°ж”ҫеҲ° values.yaml ж–Үд»¶дёӯпјҢйҖҡиҝҮдёҚеҗҢзҺҜеўғзҡ„ values ж–Үд»¶дёӯзҡ„еҸӮж•°иЎҘе……е’ҢиҰҶзӣ–пјҢе®һзҺ°еӨҡзҺҜеўғеӨҡеә”з”Ёзј–жҺ’гҖӮ

йҖҡиҝҮ helm template е‘Ҫд»ӨеҸҜд»Ҙйў„и§ҲжёІжҹ“еҮәзҡ„еҺҹз”ҹ YAMLгҖӮ

cd helm/apps-overlays
helm template . -f values/development.yaml
helm template . -f values/staging.yaml
helm template . -f values/production.yaml

дјҳзӮ№: жЁЎжқҝеҠҹиғҪејәеӨ§пјҢеҸҜд»Ҙй«ҳеәҰиҮӘз”ұең°еӨҚз”Ёд»Јз ҒгҖҒдҝ®ж”№еҸӮж•°е’ҢиҜ»еҸ–ж–Үд»¶з”ҹжҲҗй…ҚзҪ®пјҢеҸҜе°Ҷй…ҚзҪ®жҢүеә”з”Ёжү“еҢ…еҸ‘еёғеҲ° Helm Repo дёӯиҝӣиЎҢеҢ…з®ЎзҗҶпјӣ
зјәзӮ№: й«ҳеәҰиҮӘз”ұзҡ„е®ҡеҲ¶иғҪеҠӣд№ҹеҜјиҮҙдәҶй«ҳиҖҰеҗҲпјҢжЁЎжқҝеҢ–зҡ„й…ҚзҪ®д»Јз ҒдёҚеӨҹзӣҙи§ӮпјҢй…ҚзҪ®еӨұиҜҜж—¶еҪұе“ҚиҢғеӣҙеўһеӨ§дё”и°ғиҜ•йҡҫеәҰеўһеҠ гҖӮ
