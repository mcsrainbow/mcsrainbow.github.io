1. 创建交换分区文件，例如大小为4G

<!--more-->

```bash
cd /opt
touch swapfile
dd if=/dev/zero of=/opt/swapfile bs=1024 count=4194304
```

```text
4194304+0 records in
4194304+0 records out
4294967296 bytes (4.2 GB) copied, 146.071 s, 28.7 MB/s
```

```bash
ls -lh
```

```text
total 4.0G
-rw-r–r– 1 root root 4.0G 2007-3-13 15:04 swapfile
```

说明：从/dev/zero设备读取字节到/opt/swapfile文件，bs代表每块大小以k为单位，`bs=1024`表示每块1M，`count=4194304`表示`4096*1024`总共4G。

对于文件系统ext4和xfs，还可以使用`fallocate -l 4g /opt/swapfile`通过文件预留的方式来快速创建。

2. 格式化交换分区文件

```bash
mkswap -c /opt/swapfile
sync
```

说明：`mkswap -c /opt/swapfile`代表检查/opt/swapfile文件物理损坏的区域，标记后绕过坏道并进行格式化，sync确保格式化信息被写入物理文件/opt/swapfile里。

3. 激活交换分区

```bash
swapon /opt/swapfile
```

4. 修改配置文件使开机自动激活交换分区

```bash
vim /etc/fstab
```

增加以下一条记录：

```text
/opt/swapfile swap swap defaults 0 1
```

5. 最后通过`free -m`指令可查看现有交换分区的使用情况
